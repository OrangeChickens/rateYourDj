# 图标资源说明

本目录需要以下图标文件才能使小程序正常运行：

## TabBar 图标（必需）

### 首页 Tab
- `tab-home.png` - 首页图标（未选中）- 建议尺寸：81x81px
- `tab-home-active.png` - 首页图标（选中）- 建议尺寸：81x81px
- 颜色：未选中 #999999，选中 #FFD700（金色）

### 收藏 Tab
- `tab-favorite.png` - 收藏图标（未选中）- 建议尺寸：81x81px
- `tab-favorite-active.png` - 收藏图标（选中）- 建议尺寸：81x81px
- 颜色：未选中 #999999，选中 #FFD700（金色）
- 图标建议：星星或心形

### 我的 Tab
- `tab-profile.png` - 个人中心图标（未选中）- 建议尺寸：81x81px
- `tab-profile-active.png` - 个人中心图标（选中）- 建议尺寸：81x81px
- 颜色：未选中 #999999，选中 #FFD700（金色）
- 图标建议：用户头像轮廓

## 占位图片（可选但推荐）

### 默认头像
- `default-avatar.png` - 默认DJ/用户头像 - 建议尺寸：200x200px
- 颜色：灰色 #CCCCCC 背景，可以用简单的人像剪影

### 匿名头像
- `anonymous-avatar.png` - 匿名用户头像 - 建议尺寸：200x200px
- 颜色：灰色 #CCCCCC 背景，可以用问号或面具图标

## 快速创建方法

### 方法1：使用在线图标生成器
1. 访问 https://www.iconfont.cn/ 或 https://iconpark.oceanengine.com/
2. 搜索相应图标（home, star, user等）
3. 下载PNG格式，尺寸选择81x81px
4. 保存到此目录

### 方法2：使用Figma/Sketch设计
创建81x81px画布，设计简单图标：
- 首页：房子或网格图标
- 收藏：空心/实心星星
- 我的：用户圆形头像轮廓

### 方法3：临时占位（开发测试用）
可以创建纯色正方形PNG作为临时图标：
- 未选中：#999999 背景
- 选中：#FFD700 背景
- 中心放一个简单的文字或emoji

## 配色方案
- 主色（选中）：#FFD700 金黄色
- 次色（未选中）：#999999 灰色
- 背景：#FFFFFF 白色
- 文字：#000000 黑色

## 注意事项
1. 所有图标必须是PNG格式，支持透明背景
2. TabBar图标推荐使用简洁的线条图标
3. 选中和未选中状态的图标应该是同一个图标的不同颜色版本
4. 图标风格应保持一致，符合极简主义设计理念
